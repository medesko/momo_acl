# MoMo ACL

A Node.js module that manage user roles and permissions.

## Features

- Users
- Roles
- Hierarchies
- Resources
- Express middleware for protecting resources.
- Unit test coverage.

## Installation

Using npm:

```sh
npm install node_acl
```
